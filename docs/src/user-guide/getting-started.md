# Getting Started

This guide will help you get started with Inference Lab.

## Installation

Install from crates.io:

```bash
cargo install --locked inference-lab
```

Or build from source:

```bash
cargo build --release
```

## Running Your First Simulation

```bash
inference-lab -c config.toml
```
