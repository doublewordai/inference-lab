# Introduction

Welcome to the Inference Lab documentation!

## What is Inference Lab?

Inference Lab is a simulator for LLM inference workloads.

## Features

- Multiple scheduling policies (FCFS, SOF, SIF, STF, LIF, LOF, LTF)
- Real dataset support
- Prefix caching simulation
- Performance metrics and analysis
